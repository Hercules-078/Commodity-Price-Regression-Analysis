# Commodity-Price-Regression-Analysis
Streamlit app link - https://commodity-price-regression-analysis.streamlit.app/

This project analyzes global commodity price co-movements from 1995 to 2025 using correlation and linear regression. Built with Python and Streamlit, it explores relationships across Metals, Energy, and Beverages sectors through interactive visualizations and regression analysis based on reliable global data sources.

## Table of contents

1. **Abstract**: This project studies global commodity price relationships from 1995 to 2025 using correlation and linear regression. It explores how prices of Metals, Energy, and Beverages move together.

2. **Methodology**: Monthly data of 17 commodities were cleaned, analyzed, and visualized using Streamlit. Correlation and regression techniques identified strongly related pairs.

3. **Algorithm**: Pearson correlation detected high co-movement pairs, and linear regression measured their relationship strength using RÂ² values.

4. **Conclusion**: The study found notable co-movements among certain commodities, suggesting interconnected global market trends.

5. **References/Data Sources**: Data were taken from World Bank Commodity Prices, Index Mundi, MacroTrends, and RupeeRates, with references from studies on commodity price co-movements.
